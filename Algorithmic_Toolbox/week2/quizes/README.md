## Big-O Quiz ##

1. Is it true that `log_2 n = O(n^2)`
**Answer**: Yes

2. `n log_2 n = O(n)`
**Answer**: No

3. `n^2 = O(n^3)`
**Answer**: Yes

4. `n = O(sqrt(n))`
**Answer**: No

5. `5^(log_2 n) = O(n^2)`
**Answer**: No

6. `n^5 = O(2^(3 log_2 n))`
**Answer**: No

7. `2^n = O(2^(n+1))`
**Answer**: Yes
